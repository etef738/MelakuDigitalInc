# Assets Directory

## Video Background (Optional)

To add a video background to the hero section:

1. Add your video file as: `hero-bg.mp4`
2. Recommended specifications:
   - Format: MP4 (H.264 codec)
   - Resolution: 1920x1080 (Full HD)
   - Duration: 10-30 seconds (looping)
   - File size: < 5MB for optimal loading
   - Content: Abstract tech animations, particle effects, or subtle motion graphics

The website will gracefully degrade to the animated canvas network background if no video is present.

## Suggested Video Themes
- Network connections/data flow
- Particle systems
- Abstract geometric patterns
- Code/binary animations
- AI/tech visualizations
